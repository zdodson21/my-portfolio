<chat-agent> ​components

  • Developed ​AI ​Chat ​Interface ​which ​can ​be used by HAX user's to ask questions to a proprietary LLM, with the overall front end AI utilizing the name Merlin.
  • Front-end Design & Logic:
    • Chat ​logging ​system ​utilizing ​a ​single array of objects to track and display (using array mapping) all messages sent and received between the user and Merlin-AI. This system makes it simple to both display the data and download it to the user's computer as either a .txt or .json file.
    • Each ​piece ​of ​the ​interface ​is ​it's ​own ​self-contained component (chat-button, chat-developer-panel, chat-interface, chat-message, chat-suggestion, etc.). The components rely on a store file that handles state management, and the components all send and retrieve the information they need from the store using MobX.
  • Developer ​Mode
    • Users ​can ​enable ​developer ​mode ​via ​the ​appropriate button in the control bar located at the top of the interface. By doing this, users can access the following developer features:
      • Browser ​console ​logging ​of all events that happen, along with warnings and error messages.
      • Outputting ​to ​the browser console ​a ​table of messages sent by the user, by Merlin, or outputting the entire chat log.
      • Quick ​switching ​of the back end LLM, which can potentially yield different responses.
      • Quick ​switching ​of ​context ​to ​different ​courses, ​such ​as ​phys211 ​or ​Intro ​to ​Python.
      • Downloading ​chat ​log ​as ​a ​.json ​file.
